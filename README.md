# work-with-variables
разбор всех переменных

## Уровень 1: Базовое понимание

Задача 1.1 — Объявление переменных
- Объяви 3 переменные с разными типами данных (строка, число, булево)
- Присвой им значения и выведи в консоль

Задача 1.2 — Именование
- Создай переменные с правильными именами (camelCase, понятные названия)
- Создай 3 переменные с ошибочными именами (с цифрой в начале, с пробелом, ключевое слово) — прокомментируй, почему они не работают

## Уровень 2: Работа со значениями

Задача 2.1 — Изменение значений
- Объяви переменную
- Присвой значение
- Измени значение 3 раза
- Выведи каждое изменение в консоль

Задача 2.2 — Копирование переменных
- Создай переменную a со значением
- Скопируй её в переменную b
- Измени b
- Выведи обе переменные, объясни результат

Задача 2.3 — Математические операции
- Создай 2 числовые переменные
- Выполни: сложение, вычитание, умножение, деление
- Результат каждой операции сохрани в новую переменную
